- [ ] **Визуализатор агентов (Task Audit)**: Дашборд для реального времени, где юзер видит, как Morphs пишут код, пишут тесты и спорят между собой.

### Эпик 2: Cloud Sync & Backup Engine (PRO Tier)
Обеспечение защиты local-first SQLite системы.
- [ ] **Двухфазный коммит базы**: Создание бэкапа SQLite (через `tar.gz` или git staging) перед тем, как система заливает сгенерированные агентом мутации.

### Эпик 3: Изоляция Уровня 2 (Hardened Sandbox)

- [ ] **Chroot / Контейнеризация**: Полная изоляция Node/Vite/Python Sandbox-среды от хостового Макбука.
- [ ] **Network Egress Block**: Блокировка внешних сетевых вызовов из песочницы, за исключением белого списка (например, NPM Registry).
