# AI Projects Hub — оперативная память экосистемы

## Стек
- Rust / Tauri (TeleFeed, Antigravity)
- Next.js / TypeScript / React (TrendsMap Frontend, GitHub-Store)
- Python / SQLite (TrendsMap Backend, Turboquant)

## Архитектура (Проекты)
Папка `/Users/klai/AI` является хабом для всех разрабатываемых ИИ-продуктов и утилит:
- **TeleFeed**: Нативный клиент Telegram на Tauri с умной персонализированной лентой.
- **TrendsMap**: D3 Treemap агрегатор инфоповодов (Reddit, HN, TG) использующий связку Python + Next.js + Gemini.
- **GitHub-Store**: Веб-фронтенд магазина приложений и будущий Tauri лаунчер для экосистемы Antigravity.
- **kitchen-manager** / **craft-house** / **antigravity-bar**: Компоненты инфраструктуры Antigravity.
- **turboquant_mlx**: Проекты ML/PyTorch, вероятнее всего эксперименты с квантованием/памятью.
- **local-ai-lab**: Универсальная лаборатория для тестирования локальных моделей (Gemma 4, Llama, Mistral) через MLX/Ollama.

## Ключевые решения
- Мы являемся частью продвинутой среды разработки агентов от Google Deepmind.
- Каждая папка проекта содержит собственный `GEMINI.md` (живую память), который нужно читать при старте для моментального погружения. В этом же хабовом `GEMINI.md` мы будем хранить общие полезные материалы для всех проектов.
- **АВТОБЛОГГИНГ (Craft House)**: После завершения любого майлстона, релиза публичной фичи или крутого архитектурного эксперимента, ИИ **ОБЯЗАН** запустить системный воркфлоу `publish_blog` (`/Users/klai/.gemini/antigravity/workflows/publish_blog.md`), чтобы автоматически опубликовать безопасный и творческий пост в Telegram-канал Craft House Agent Feed.
- **СТАНДАРТ НОВЫХ ПРОЕКТОВ (Локальная .gemini)**: Мы перешли от монолитного `GEMINI.md` к версионируемой папке `.gemini/` внутри проектов. `GEMINI.md` остаётся "точкой входа", но: 1) логика переезжает в `.gemini/rules/` (стандарты), `.gemini/workflows/` (скрипты), и 2) настройки среды ложатся в `.gemini/context.json` (МСР, порты, пути к БД).
