# Apps (Clients & Backoffice)

Здесь хранятся основные клиентские приложения ресторана в архитектуре Turborepo.

- **`backoffice/`** - Приложение для компьютеров и планшетов (Tauri v2 + React). Offline-First решение для управления чеками, инвентаризацией и рецептами.
- **`landing/`** - Витрина магазина, интерфейс для выбора и заказа блюд клиентами (Next.js App Router).

Каждое приложение может иметь свой `package.json` и настраивается независимо, но делит общие компоненты из папки `packages/`.
