# Backend (PocketBase Hooks)

В этой директории находятся кастомные скрипты для облачного PocketBase бэкенда.

## Описание
Мы используем Self-Hosted инстанс `PocketBase` развернутый на `borsch.shop`.
Он выполняет роль центрального "Source of Truth" для мобильных приложений (Next.js) и оффлайн-клиентов админок (Tauri).

- `/pb_hooks` - Содержит JavaScript-хуки, которые PocketBase подхватывает автоматически.
  - `main.pb.js` - триггеры после добавления/изменения данных (например: рассылка заказов в Telegram).
  - `cron_archive.pb.js` - фоновые задачи (например: автоматическая архивация заказов за день каждые 5 минут).

## Загрузка в облако
Для обновления хуков на рабочем сервере, файлы из `pb_hooks` необходимо скопировать в директорию PocketBase на вашем VPS (например, `scp -r pb_hooks/* root@borsch.shop:/path/to/pb_data/pb_hooks/`).
PocketBase автоматически перезагружает хуки при их изменении, без необходимости перезапуска сервиса.
