# Blueprint: Локальная Оффлайн-Касса (POS)

## Описание
Заготовка для бизнеса. POS терминал для кафе или розницы.

## Базовые фичи (Что внедрить в ТЗ разработчику-Архитектору):
1. Корзина товаров (локальный state, сохраняется в SQLite/IndexedDB на случай обрыва сети).
2. Чек-аут (посчитать сумму + налоги).
3. Экран ввода пинкода (кассир) или выбора столика.
4. Синхронизация: Заказы отправляются на сервер только при наличии интернета.
