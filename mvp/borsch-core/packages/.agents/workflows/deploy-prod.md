---
description: Автоматизированный чеклист для безопасного деплоя проекта на Staging/Prod сервера.
---

0. **СИСТЕМНОЕ:** Строго первым делом немедленно выведи в чат: `🔄 **Контекст:** [Агент: Current] | [Скилл: None] | [Воркфлоу: /deploy-prod]`

1. Запроси у пользователя цель деплоя (какой проект, какая ветка).
2. Вызови агента `devops`.
3. Запусти все тесты (юнит/интеграционные). Если падают — прерви процесс.
4. Убедись, что Docker образы или сборка (npm run build / cargo build --release) проходят успешно.
5. Залей бинарники / статику на сервер или запусти скрипт синхронизации (см. схемы `alpha-feed` или `borsch-shop`).
6. Сообщи об успехе и выведи ссылки на логи платформы.
