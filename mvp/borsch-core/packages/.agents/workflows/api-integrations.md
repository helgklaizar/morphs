# API Integration & Agent Setup

**Когда использовать:** Если агенту нужно дать доступ к 3rd-party сервису со Swagger/OpenAPI спецификацией (например, база данных Supabase REST, платежи Stripe, внутренняя аналитика).

## Правила работы ИИ:
1. **Никаких кастомных MCP**-серверов! Они жрут контекст и падают при ошибках парсинга JSON.
2. Взаимодействуй с API через нативную консоль. Используй пакет `openapi-to-cli`.

### Шаблон настройки:
```bash
# Генерация нативного CLI
npx openapi-to-cli@latest [название-cli] [ссылка-на-много-swagger-контракт.json]

# Тестирование прямо в bash
[название-cli] [endpoint_name] --param1 value
```
3. Для всех будущих вызовов этого API, просто генерируй bash-команды `[название-cli]`, получай чистый JSON-ответ в stdout и работай с ним без прослоек.
