# Borsch Core

Minimalist order and pos system. Forked from the original `borsch-shop` monorepo.

## Project Structure
- `apps/landing` - Next.js Public Web App
- `apps/backoffice` - Next.js & Tauri Desktop App (POS system)
- `packages/db` - Database and API types
- `packages/types` - Shared TS interfaces
