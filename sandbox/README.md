# Sandbox (Песочница)

Изолированная среда выполнения сборок и тестов "Морфов". 

## Роль в системе
Здесь генерируется код от ИИ, а затем прогоняется через систему сборки и тестирования (Vitest). Код не прокидывается дальше или возвращается в Конфигуратор без прохождения всех тестов в Sandbox (подход Test-Driven Mutation).

## Стек
- Node.js
- React
- Vitest для тестов

## Запуск тестов песочницы
```bash
npm install
npm run test
```
